# my-quickshell-project

A personal status bar for Wayland on Arch Linux, built using quickshell.

This project is heavily work in progress and is not intended to be used by others as-is.

---

## Features

### Currently Implemented

- Clock widget
- Workspace indicator with switching functionality
- Now playing widget for displaying current media
- CPU, GPU, and RAM utilization
- Network usage (currently only for the default interface)
- Update checker widget
- Notification popups and notification list
- System tray icons
- Volume indicator with global and per-application audio controls
- Ability to change active audio input and output devices
- Menu for shutdown, reboot, lock, and logout

---

## Work in Progress / TODO
